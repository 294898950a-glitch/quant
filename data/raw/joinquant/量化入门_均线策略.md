# 量化投资入门：均线策略详解

## 概述
均线（Moving Average, MA）是最基础也是最经典的量化技术指标之一。它通过计算一段时间内的平均价格来平滑短期波动，识别趋势方向。

## 核心概念

### 简单移动平均线 (SMA)
SMA = (P1 + P2 + ... + Pn) / n

其中 P 为价格，n 为时间窗口。常用的周期有 5日（周线）、20日月线）、60日（季线）。

### 指数移动平均线 (EMA)
EMA 今日 = α × P今日 + (1-α) × EMA昨日
α = 2 / (n + 1)

EMA 对近期价格赋予更高权重，反应更灵敏。

## 经典均线策略

### 1. 双均线交叉策略（Golden Cross / Death Cross）
- **买入信号**：短期均线上穿长期均线（金叉）
- **卖出信号**：短期均线下穿长期均线（死叉）
- **常用参数**：MA5/MA20, MA10/MA30, MA20/MA60

### 2. 均线多头排列
短、中、长期均线从上到下依次排列，趋势向上。

### 3. 均线支撑/阻力
价格回调到均线位置获得支撑（上升趋势）或遇到阻力（下降趋势）。

## 策略优缺点

### 优点
- 计算简单，易于理解和实现
- 在趋势行情中表现良好
- 能有效过滤噪音

### 缺点
- 震荡市中频繁产生假信号（whipsaw）
- 存在滞后性（lag），参数越大滞后越明显
- 单一均线策略收益有限，需要结合其他指标

## 实际应用建议
1. 结合成交量确认信号有效性
2. 加入止损机制控制回撤
3. 不同市场周期可能需要不同参数
4. 建议与动量指标或波动率指标组合使用

## 相关指标
- MACD（基于EMA的衍生指标）
- BOLL（布林带，基于标准差和均线）
- RSI（相对强弱指数）
- ATR（真实波幅）
