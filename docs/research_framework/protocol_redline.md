## 红线段

```text
<!-- protocol-redline-v1.5 -->
R1 入口: 新会话只先读 3 个入口文件: CURRENT.md / baseline_registry.md / docs/INDEX.md.
R2 证据: 结论必须指向数据、报告、记录、输出或 git 事实; 不能凭感觉编.

R4 预算: 新数据、研究代码、研究参数可自动做; 先用 estimate_compute_budget.py 算预算, ≤ ¥100 可继续, > ¥100 或算不出才等用户.
R5 保护: 不自动归档当前策略, 不自动升级实盘, 不自动改协议本身, 不碰已确认无效方向.
R6 真值: 新成绩、状态变化、立项/撤项、关键提升必须同步 CURRENT.md + baseline_registry.md; 回测完成必须写 run_manifest.
R7 通信: outbox 方向不能写反; 超过 10 分钟要报进度; 已处理消息不能重复抢占.
```

## 当前执行规则

### R1 入口

新会话只能先读:
1. `docs/research_framework/CURRENT.md`
2. `data/research_framework/baseline_registry.md`
3. `docs/INDEX.md`

其他文档都是非入口文件。需要查原因时再读，不作为当前状态第一来源。

### R2 证据

有效结论必须能指向:
- 数据文件
- 报告
- 运行输出
- 记录文件
- git 事实
- outbox/state 消息

没有证据只能写“待验证”，不能写成结论。

### R3 研究设计

新研究 batch 不能手工拼文件夹。必须从 `scripts/new_research.py` 开始。

跑批前必须过 5 道闸:
1. 研究设计结构
2. 数据字段
3. 预算配置
4. 研究合理性
5. 跑批脚本接入检查

跑完回测或评估 batch 必须写 run manifest。

### R4 预算和执行

用户已授权:
- 新数据可以自动接入
- 研究代码可以自动修改
- 研究参数可以自动增加
- 预算计算结果 ≤ ¥100 时可以自动继续

必须等待用户:
- 预算 > ¥100
- 预算算不出来
- 需要改协议红线本身
- 需要碰“已确认无效”方向
- 需要把当前策略归档
- 需要升级实盘

远端或 spot 不是单独红线。是否能跑只看预算计算结果和研究设计是否通过。

### R5 保护边界

允许自动改:
- 研究脚本
- 评估脚本
- 新数据处理脚本
- 原型策略逻辑
- 研究参数空间

不能自动做:
- 把原型提升为当前主策略真值
- 把策略标成可实盘
- 永久归档当前策略
- 改协议红线
- 复活已确认无效方向

生产主路径的核心定价、成本、切分逻辑如果只是研究原型, 可以改; 如果要替换当前主策略或成为实盘依据, 必须用户拍板。

### R6 真值同步

触发同步:
- 新 baseline 数字
- 状态变化
- 新策略立项
- 撤销立项
- 关键文件从研究原型提升为当前依据

必须同步:
- `docs/research_framework/CURRENT.md`
- `data/research_framework/baseline_registry.md`
- 必要时同步 run manifest

不触发:
- 单纯读文件
- 单纯查格式
- 单纯估预算
- 单纯 ACK / PROGRESS

### R7 通信

outbox 方向:
- `claude/outbox.md`: Claude 写给 Codex
- `codex/outbox.md`: Codex 写给 Claude

超过 10 分钟的任务必须写进度。

收到消息前先查是否已经 ACK/RUNNING/processed, 不能重复启动同一个任务。

## 30 分钟无回复默认

当系统问用户下一步, 30 分钟内没有回复:

1. AI 先读 3 个入口文件。
2. AI 填写 `当前策略`。
3. 保持 `当前策略`。
4. 换一个研究方向。
5. 不重复刚失败的方向。
6. 不归档。
7. 不升级实盘。
8. 新数据、研究代码、研究参数可继续。
9. 预算由 `scripts/estimate_compute_budget.py` 计算, ≤ ¥100 继续, > ¥100 或算不出才等用户。

## 模式 B 自动执行边界

模式 B 可以自动做:
- 按经验账本选下一个方向
- 写设计
- 接新数据
- 改研究代码
- 加研究参数
- 跑评估
- 写报告和记录

模式 B 必须停止:
- 预算 > ¥100
- 预算算不出来
- 轮数达到上限
- 用户回来发消息
- 可探索方向为空
- 需要改协议
- 需要碰已确认无效方向
- 需要归档或实盘升级
